# Synth Grid

A web-based music synthesizer with an interactive grid interface for composing and playing tracks.

**Live Demo:** [synth-grid.zeeshanyousuf.io](https://synth-grid.zeeshanyousuf.io)

## About

Create music by clicking cells on an interactive grid. Features real-time audio visualization on canvas, a clean dark UI, and multiple sound options. No musical knowledge required — just click and play.

## Tech Stack

- Node.js (server)
- Web Audio API
- HTML5 Canvas
- Vanilla JavaScript

## Run Locally

```bash
node server.js
```

Then open `http://localhost:3001` in your browser.
